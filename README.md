## 🚀 How to Use

```bash
# 1. Clone the repository
git clone <repo-url>

# 2. Switch to a branch
git checkout java-fundamentals

# 3. Open in your IDE (IntelliJ / VS Code)
# 4. Run any .java file
javac FileName.java
java FileName
```

> 💡 **Tip:** Follow the branches in order — `Java Fundamentals → Java Core → OOPS → Java Practice` — for the best learning progression.
